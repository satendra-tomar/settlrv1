# Settlr Brand Kit v1.0

Production asset pack generated from the supplied final approved Settlr logo source.

## Source

- Original source: `source/settlr logo 1.jpg`
- Extracted app mark: `source/logo-source-extracted.png`
- Extracted foreground mark: `source/logo-foreground-source.png`
- Figma reference file created: https://www.figma.com/design/2po431e2PokGTJym9Xlttw

The source file is a flattened JPG brand board, not editable vector path artwork. To avoid redesigning, tracing, or reinterpreting the approved logo, SVG/PDF/EPS exports preserve source-derived artwork rather than fabricating new editable geometry.

## Export Notes

- PNG exports use transparent backgrounds where required and high-quality Lanczos resampling.
- iOS marketing and store icons are exported without alpha for App Store compatibility.
- Android adaptive foreground contains only the extracted route mark; adaptive background uses the original Settlr branding color `#2E1065`.
- Source crop bounding box from the provided image: `(184, 230, 559, 605)`.

## Expo

Use `app.json` in this kit or merge its `expo` object into your existing Expo config.
